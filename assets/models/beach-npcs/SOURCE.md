# Kenney beach NPC provenance

The Beach realm uses five distinct animated GLB characters from two Kenney packs:

- [Mini Skate 1.2](https://kenney.nl/assets/mini-skate): `character-skate-girl.glb` and `character-skate-boy.glb`.
- [Mini Characters 1.0](https://kenney.nl/assets/mini-characters): `character-female-a.glb`, `character-female-c.glb`, and `character-male-b.glb`.

Both official pack pages declare the assets **Creative Commons CC0 1.0** and both downloaded archives include Kenney's `License.txt`. Each model is kept unmodified with its adjacent `Textures/colormap.png` dependency. The Beach runtime plays the authored `idle` clip through one `THREE.AnimationMixer` per unique model.

Official archive URLs used on 2026-07-13:

- `https://kenney.nl/media/pages/assets/mini-skate/00b0c2b304-1709221152/kenney_mini-skate.zip`
- `https://kenney.nl/media/pages/assets/mini-characters/bfc7e272b4-1774770718/kenney_mini-characters.zip`

The source archives are not checked in; only the selected runtime files are included.
