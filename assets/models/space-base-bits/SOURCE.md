# Space Base Bits asset selection

Source archive: `KayKit_Space_Base_Bits_1.0_FREE.zip` (version 1.0)

This project keeps only the GLTF, BIN, and shared texture files used by the Space Realm. Custom base buildings and the pack's building modules are intentionally excluded for the current revision.

Included categories: landing pads, landers, mining equipment, power equipment, lights, trucks, cargo, containers, rocks, utility towers, and a road tunnel.

The Space Realm currently references 26 reusable source models. Base-module and cargo-depot building assets remain intentionally excluded.
