# KayKit Furniture Bits

- Downloaded: 2026-07-13

This directory contains only the glTF models used by My House, their matching
binary buffers, and the pack's shared atlas texture. The included
`LICENSE-CC0.txt` is the license file distributed with the pack.

The game retains its Three.js primitive fallbacks and uses them whenever a
model cannot load. Unmapped furniture such as the TV, refrigerator, desk,
vanity, and bookshelf intentionally remains project-authored.
