# Quaternius animated modular humanoid

- Source: **Ultimate Modular Men Pack** by Quaternius
- Official page: https://quaternius.com/packs/ultimatemodularcharacters.html
- Official download folder: https://drive.google.com/drive/folders/1USAAquX2JJWuA2m6zol0KUkFe3UkZ8zX
- License: **CC0 1.0 Universal** — https://creativecommons.org/publicdomain/zero/1.0/
- Source model: `Individual Characters/FBX/Casual_Hoodie.fbx`

The official pack describes 11 characters, four swappable model sections per character,
and 24 animations. It permits personal and commercial use under CC0.

## Runtime conversion

The FBX was converted locally to binary glTF with the open-source Meta
`FBX2glTF` converter, then reduced to the three locomotion clips used by this
game. Geometry, skinning, named materials, and animation data are unchanged.

Runtime asset: `quaternius-casual-humanoid.glb`

- Size: 1,144,160 bytes
- Render vertices: 18,618
- Uploaded vertices: 12,518
- Material customization regions: `Skin`, `Hair`, `Purple`, `LightBlue`
- Retained clips: `CharacterArmature|Idle`, `CharacterArmature|Walk`,
  `CharacterArmature|Run`
