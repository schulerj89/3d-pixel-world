The repository includes the original embedded glTF exports for
`Enemy_ExtraSmall` and `Enemy_Small`. Both models retain their authored
`Flying_Idle` animation clip. Geometry, skinning, materials, textures, and
animation data are embedded in each glTF, so no conversion or generated
content is involved.

Only these two reusable variants are shipped from the 92-model source pack to
keep the Space Realm payload small. Runtime instances share model resources,
use skeleton-safe clones, and are grounded from measured model bounds.
